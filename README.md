# Data Analysis Portfolio

Source files for a static portfolio website featuring data analysis and web application projects.

## Website

[Open the portfolio](https://sintagmatarches.github.io/portfolio/)

## Projects

- [European Songbook](https://european-songbook-portfolio.pages.dev/) — a public demo of a multilingual historical song catalog with search, filters, map navigation, and simulated administration workflows. The demo code is available in the [showcase repository](https://github.com/Sintagmatarches/european-songbook-showcase).
- [E-commerce Funnel Analysis](https://sintagmatarches.github.io/portfolio/assets/case-study-1-ecommerce-funnel.pdf) — an event-level funnel analysis based on theLook dataset, using BigQuery and Power BI.
- Regional data visualizations — comparisons of heating prices and median salaries by county.

## Repository contents

- `index.html` — site content and structure;
- `styles.css` — layout and visual styles;
- `script.js` — client-side behavior;
- `assets/` — reports and visualizations.
