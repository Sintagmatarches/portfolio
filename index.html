01

BigQuery · SQL · Power BI

E-commerce Funnel Analysis

An event-level analysis of theLook e-commerce data to identify the largest conversion loss and the markets driving the absolute gap.

Question
Where does the product-to-purchase funnel lose the most volume, and which geographic segments should be investigated first?
Data
843,608 product events, 593,628 cart events, and 181,420 purchase events from theLook dataset.
Analysis
Built an event-level funnel, calculated stage conversion, and segmented absolute loss by country and city.
Result
The largest drop occurs after cart: 412,208 events, with 30.56% cart-to-purchase conversion. China, the United States, and Brazil account for the largest absolute losses.
Recommendation
Rebuild the funnel at user or session level with chronological steps, then test checkout, shipping, payment, and confirmation events in the highest-volume markets.

02

Data Product · Search · Cloudflare

European Historical Songbook

A multilingual data product for exploring a structured historical song corpus through catalog search, metadata filters, and map navigation.

Question
How can a large historical corpus remain searchable and understandable across languages, periods, countries, and authorship metadata?
Data
Structured song and authorship metadata. The public showcase uses 305 sanitized demo entries with a strict production-data boundary.
Build
Implemented full-text retrieval, multidimensional filtering, song pages, map navigation, and simulated administration workflows.
Result
Delivered a runnable public demonstration with an isolated API, no production credentials, and automated checks for data and interface boundaries.
Recommendation
Keep public and production data paths explicitly separated, and validate search quality as the catalog and metadata model grow.
